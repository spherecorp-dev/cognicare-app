# Meta Ads Specifications

## Creative Specifications

### Image Ads
- **Recommended size:** 1200 x 628px
- **Aspect ratio:** 1.91:1 (landscape) or 1:1 (square)
- **File types:** JPG, PNG
- **Max file size:** 30MB
- **Text in image:** Minimize, avoid >20% text

### Video Ads
- **Recommended resolution:** 1080 x 1080px (1:1) or 1920 x 1080px (16:9)
- **Aspect ratio:** 1:1, 4:5, 9:16, 16:9
- **File types:** MP4, MOV
- **Max file size:** 4GB
- **Duration:** 1 second to 241 minutes (recommended: 15-60s)
- **Frame rate:** 30fps minimum

## Copy Specifications

### Primary Text
- **Character limit:** 125 characters (recommended for mobile)
- **Max characters:** 65,000

### Headline
- **Character limit:** 27 characters (recommended)
- **Max characters:** 255

### Description
- **Character limit:** 27 characters (recommended)
- **Max characters:** 30

### Call-to-Action
- Standard options: Learn More, Shop Now, Sign Up, Download, etc.

## Campaign Structure

### CBO (Campaign Budget Optimization)
- **Use when:** Testing multiple audiences, scaling
- **Budget:** Set at campaign level
- **Benefits:** Automatic budget distribution to best-performing adsets

### ABO (Ad Set Budget Optimization)
- **Use when:** Need precise control per audience
- **Budget:** Set at adset level
- **Benefits:** More control over spend per audience

## Bidding Strategies by Creative Profile

### Blackhat DR (Direct Response)
- **Strategy:** Cost Cap
- **Bid amount:** 1.5x target_cpa initially
- **Optimization goal:** Conversions
- **Notes:** Aggressive scaling, high creative turnover

### Whitehat
- **Strategy:** Lowest Cost or Bid Cap
- **Bid amount:** 1.2x target_cpa
- **Optimization goal:** Conversions or Value
- **Notes:** Sustainable scaling, brand safety

### SaaS
- **Strategy:** Lowest Cost
- **Optimization goal:** Lead or Link Click
- **Notes:** Longer sales cycles, focus on qualified leads

## Platform Limits

### Daily Spend Limits
- New account: $50-250/day
- Warmed account: $1,000+/day
- Established account: $10,000+/day

### Creative Limits
- **Max ads per adset:** 50 (recommended: 6-10)
- **Max adsets per campaign:** 70
- **Max campaigns per account:** No limit

## Creative Approval

### Common Rejection Reasons
1. Too much text in image (>20%)
2. Misleading claims
3. Prohibited content (weapons, drugs, etc.)
4. Poor landing page experience
5. Broken links

### Appeal Process
1. Review rejection reason
2. Fix issue
3. Resubmit or request review
4. Wait 24-48h for response
