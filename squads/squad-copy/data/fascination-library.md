## Formatos de Fascination

> Cada formato e um template. Substituir [placeholders] com dados especificos da oferta.

### Formato 1: "Como... sem..."

Promete resultado removendo objecao principal.

```
EN: "How to [resultado desejado] without [sacrificio temido]..."
FR: "Comment [resultado desejado] sans [sacrificio temido]..."
ES: "Como [resultado desejado] sin [sacrificio temido]..."
```

### Formato 2: "Por que [crenca] esta errada..."

Invalida solucao anterior, cria abertura para novo mecanismo.

```
EN: "Why [crenca popular] is actually making things WORSE (and what to do instead)..."
FR: "Pourquoi [crenca popular] empire en realite les choses (et quoi faire a la place)..."
ES: "Por que [crenca popular] en realidad esta empeorando las cosas (y que hacer en su lugar)..."
```

### Formato 3: "O [substancia/ritual] de [tempo]..."

Especificidade + baixa barreira de entrada.

```
EN: "The [tempo] [ritual/substancia] that [resultado]..."
FR: "Le [ritual/substancia] de [tempo] qui [resultado]..."
ES: "El [ritual/substancia] de [tempo] que [resultado]..."
```

### Formato 4: "A verdade sobre [X] que [autoridade] nao quer..."

Forbidden knowledge — cria tensao com autoridade.

```
EN: "The truth about [topico] that [autoridade] doesn't want you to know..."
FR: "La verite sur [topico] que [autoridade] ne veut pas que vous sachiez..."
ES: "La verdad sobre [topico] que [autoridade] no quiere que sepas..."
```

### Formato 5: "[Numero] [pessoas] ja descobriram..."

Social proof quantificado.

```
EN: "[Numero] [tipo de pessoa] have already discovered the secret to [resultado]..."
FR: "[Numero] [tipo de pessoa] ont deja decouvert le secret pour [resultado]..."
ES: "[Numero] [tipo de pessoa] ya descubrieron el secreto para [resultado]..."
```

## Regras de Uso

1. **NUNCA copiar literalmente** — Usar como modelo de ESTRUTURA, adaptar com mecanismo e angulo da oferta
2. **Respeitar compliance** — Trocar claims diretos por linguagem de substituicao (ver agent stefan-georgi.md)
3. **Adaptar ao geo** — Exemplos EN sao mais agressivos. Suavizar para FR. Aquecer para ES.
4. **Cada fascination abre UM loop** — Nao tentar resolver no ad. O loop fecha no destino.
5. **Misturar formatos** — Nao usar 5 fascinations do mesmo formato. Variar entre Como/Por que/O ritual/etc.
6. **Validar com @copy-chief** — Fascinations que mencionam mecanismo devem estar alinhadas com o UMP da oferta
