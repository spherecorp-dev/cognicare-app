## User Story

**As a** Squad Orchestration Engine,
**I want** implementar pause/resume de execuções com salvamento de estado completo em File System,
**so that** posso pausar execuções gracefully, preservar estado completo, e retomar execuções sem perda de dados.

---

## Description

Este story implementa o StateManager, componente responsável por pause/resume de execuções de squads. O StateManager deve:

1. Pausar execução gracefully (espera task atual completar antes de pausar)
2. Salvar estado completo em `.aios/squad-runs/{runId}/state.yaml`
3. Validar integridade de state.yaml antes de resume
4. Retomar execução do ponto exato onde parou
5. Suportar múltiplos runs simultâneos sem conflito
6. Tratar casos de edge (state corrupto, resume de run já completo, etc.)

**Contexto Técnico:**

O StateManager garante continuidade de execução ao:
- Salvar contexto acumulado (inputs, outputs de cada step)
- Salvar índice do step atual (`currentStepIndex`)
- Salvar status de execução (`running`, `paused`, `completed`, `failed`)
- Validar checksum de state.yaml antes de resume
- Bloquear resume se estado inconsistente

**Integração com Sistema Existente:**

- Integra com `SquadOrchestrator` (Story 1.1) para pause/resume via dashboard
- Integra com `TaskExecutor` (Story 1.2) para snapshot de contexto
- Novo componente: `.aios-core/core/orchestration/squad-engine/state-manager.js`

## Integration Verification

### IV1: Pause durante execução aguarda task completar antes de salvar estado

**Test:** Graceful pause validation

```javascript
const stateManager = new StateManager(runId);
const orchestrator = new SquadOrchestrator();

// Start execution
await orchestrator.startRun(runId, squadId);

// Request pause while step 2 is executing
await stateManager.pause();

// Assert: state saved AFTER step 2 completes
const state = await stateManager.loadState();
assert(state.currentStepIndex === 2);
assert(state.status === 'paused');
```

### IV2: Resume retoma execução do ponto exato onde parou

**Test:** Resume validation

```javascript
// Load paused state
const state = await stateManager.loadState();
assert(state.status === 'paused');
assert(state.currentStepIndex === 2);

// Resume execution
await stateManager.resume();

// Assert: execution continues from step 3
const updatedState = await stateManager.loadState();
assert(updatedState.currentStepIndex > 2);
assert(updatedState.status === 'running' || 'completed');
```

### IV3: State checksum validation bloqueia resume se state.yaml corrompido

**Test:** Corruption detection

```javascript
// Corrupt state.yaml manually
const statePath = `.aios/squad-runs/${runId}/state.yaml`;
const corruptedState = await fs.readFile(statePath, 'utf-8');
await fs.writeFile(statePath, corruptedState + '\nmalicious_data: true');

// Attempt resume
try {
  await stateManager.resume();
  assert.fail('Should have thrown StateCorruptionError');
} catch (error) {
  assert(error.name === 'StateCorruptionError');
  assert(error.message.includes('Checksum validation failed'));
}
```

### IV4: Múltiplos runs simultâneos não conflitam (isolamento de estado)

**Test:** Multi-run isolation

```javascript
const run1 = new StateManager('run-001');
const run2 = new StateManager('run-002');

// Pause run1
await run1.pause();
const state1 = await run1.loadState();

// Pause run2
await run2.pause();
const state2 = await run2.loadState();

// Assert: each run has independent state
assert(state1.runId === 'run-001');
assert(state2.runId === 'run-002');
assert(state1.context !== state2.context);
```

### State.yaml Schema

```yaml
runId: string                  # Unique run identifier
squadId: string                # Squad being executed
status: enum                   # running | paused | completed | failed
currentStepIndex: number       # Index of current/next step to execute
startedAt: ISO8601             # Execution start timestamp
pausedAt: ISO8601?             # Pause timestamp (if paused)
completedAt: ISO8601?          # Completion timestamp (if completed)
context:                       # Accumulated context from all steps
  step-id-1:
    input: object
    output: object
  step-id-2:
    input: object
    output: object
metadata:
  squadVersion: string         # Squad.yaml version (for compatibility)
  engineVersion: string        # SquadOrchestrator version
  totalSteps: number           # Total steps in pipeline
  completedSteps: number       # Steps completed so far
checksum: string               # SHA256 hash of state (excluding checksum field)
```

### Graceful Pause Logic

**Pause Flow:**
```javascript
async pause() {
  // 1. Set pause flag
  this.pauseRequested = true;

  // 2. Wait for current task to complete
  await this.waitForTaskCompletion();

  // 3. Snapshot current state
  const state = this.buildState();

  // 4. Compute checksum
  state.checksum = this.computeChecksum(state);

  // 5. Save to file
  await this.saveState(state);

  // 6. Update status
  state.status = 'paused';
  state.pausedAt = new Date().toISOString();
}
```

### Checksum Validation

**Purpose:** Detect state.yaml corruption (manual edits, disk errors)

**Algorithm:**
```javascript
computeChecksum(state) {
  const { checksum, ...stateWithoutChecksum } = state;
  const stateString = JSON.stringify(stateWithoutChecksum);
  return crypto.createHash('sha256').update(stateString).digest('hex');
}

validateChecksum(state) {
  const expectedChecksum = this.computeChecksum(state);
  if (state.checksum !== expectedChecksum) {
    throw new StateCorruptionError('Checksum validation failed');
  }
}
```

### Error Handling

**StateCorruptionError:**
```javascript
class StateCorruptionError extends Error {
  constructor(message, details) {
    super(`State corruption detected: ${message}`);
    this.name = 'StateCorruptionError';
    this.details = details;
  }
}
```

**InvalidStateError:**
```javascript
class InvalidStateError extends Error {
  constructor(validationErrors) {
    super(`Invalid state: ${validationErrors.join(', ')}`);
    this.name = 'InvalidStateError';
    this.validationErrors = validationErrors;
  }
}
```

### Integration with SquadOrchestrator

**Modified SquadOrchestrator.executeRun():**
```javascript
async executeRun(runId, squadId) {
  const stateManager = new StateManager(runId);

  // 1. Check if resuming from paused state
  if (await stateManager.hasState() && await stateManager.isPaused()) {
    const state = await stateManager.loadState();
    await this.resumeFromState(state);
    return;
  }

  // 2. Normal execution (new run)
  for (let i = 0; i < steps.length; i++) {
    // Check pause flag before each step
    if (stateManager.isPauseRequested()) {
      await stateManager.pause();
      return;
    }

    await this.executeStep(steps[i], context);
  }
}
```

if (await stateManager.isPaused()) {
