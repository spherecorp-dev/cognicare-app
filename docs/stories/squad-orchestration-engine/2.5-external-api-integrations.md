## Story

**As a** Squad Orchestration Engine,
**I want** integrações diretas com APIs externas críticas (image generation, transcription, spy scraping),
**so that** tasks de squad-copy possam gerar imagens, transcrever vídeos e fazer spy scraping automaticamente.

## Acceptance Criteria

1. **AC1 — ImageGeneratorClient:** `ImageGeneratorClient` em `.aios-core/core/integrations/image-generators/client.js` com suporte para múltiplos providers:
   - DALL-E (OpenAI API)
   - MidJourney (via proxy ou API oficial)
   - NanoBanana (custom API)
   - Flux (Replicate ou similar)

2. **AC2 — Image Generator Circuit Breaker:** Circuit breaker por provider usando `APICircuitBreaker` (Story 2.3). Threshold: 5 falhas, timeout: 60s. Cada provider tem breaker independente.

3. **AC3 — Image Generator Retry:** Retry com exponential backoff (Story 2.4) para rate limits: 3x com delays 2s, 4s, 8s. Content policy errors (400): sanitize prompt e retry 1x.

4. **AC4 — Provider Fallback:** Se provider primário falha após retries, tenta provider secundário (ex: DALL-E falha → MidJourney). Fallback chain configurável no playbook ou runtime override.

5. **AC5 — WhisperClient:** `WhisperClient` em `.aios-core/core/integrations/whisper/client.js` para transcrição de áudio/vídeo:
   - Suporte multi-idioma (FR, ES, EN, auto-detect)
   - Timeout de 60s por arquivo
   - Retry 2x com timeout maior (60s → 90s → 120s) se timeout

6. **AC6 — Spy Scraping Clients:** Clients para download de ads em `.aios-core/core/integrations/spy-scraping/`:
   - `MetaAdLibraryClient` — Download de ads da Meta Ad Library API
   - `TikTokCreativeClient` — Download de vídeos do TikTok Creative Center
   - Concurrency control: max 5 downloads simultâneos (semaphore)
   - Save raw media em `.aios/squad-runs/{runId}/spy-downloads/`

7. **AC7 — Configuration:** API keys lidas de `.env`:
   - `OPENAI_API_KEY` (DALL-E + Whisper)
   - `MIDJOURNEY_API_KEY`
   - `NANOBANANA_API_KEY`
   - Provider selection configurável em playbook ou runtime override

8. **AC8 — Error Logging:** Logs estruturados em `.aios/squad-runs/{runId}/logs/api-errors.jsonl`. Cada entry: `{ timestamp, provider, error_type, status_code, message, retry_attempt }`.

9. **AC9 — Graceful Degradation:** Se API falha após todos os retries e fallbacks, step é marcado como `skipped` (não `failed`) e pipeline continua. Log warning emitido.

10. **AC10 — Dashboard Visibility:** Circuit breaker status visível via endpoint `/api/system/circuit-breakers` (Story 2.3). API error logs acessíveis via `/api/runs/{runId}/logs` (Story 1.5).

### Concurrency Semaphore Pattern

```javascript
class Semaphore {
  constructor(maxConcurrent = 5) {
    this.maxConcurrent = maxConcurrent;
    this.running = 0;
    this.queue = [];
  }

  async acquire() {
    if (this.running < this.maxConcurrent) {
      this.running++;
      return;
    }
    return new Promise(resolve => this.queue.push(resolve));
  }

  release() {
    this.running--;
    if (this.queue.length > 0) {
      this.running++;
      this.queue.shift()();
    }
  }
}
```

### Risks

- **API availability:** MidJourney, NanoBanana, Flux APIs may not have official endpoints. Mitigação: Stub implementations with defined interface. Real implementation when APIs available.
- **Large file downloads:** Spy scraping can download many large video files. Mitigação: Concurrency semaphore (max 5) + configurable download limits.
- **API costs:** Real API calls in dev could be expensive. Mitigação: All tests use mocked APIs. Real calls only in production.
- **Rate limits:** Each provider has different rate limits. Mitigação: Per-provider circuit breaker + retry with exponential backoff.
- **Content policy:** DALL-E may reject prompts with certain content. Mitigação: Prompt sanitization + retry 1x with sanitized prompt.
